# Prompt: triar inbox

```
Pegue um item de `docs/brain/99_INBOX/` e:

1. Decida o tipo: módulo, regra, decisão, open loop, aprendizado, ideia para roadmap, ou descarte.
2. Crie nota apropriada usando o template correspondente em `docs/brain/90_TEMPLATES/`.
3. Atualize [[MOC]] e o índice da pasta destino.
4. Apague o arquivo da inbox (a ideia migrou).

Se a ideia ainda for vaga, deixe um comentário em [[OPEN_LOOPS]] em vez de criar nota prematura.

ITEM:
{{conteúdo da ideia}}
```
