# 💡 INBOX

Use [[TEMPLATE_IDEIA_BRUTA]] para registrar ideias rápidas. Cada arquivo `INBOX_<slug>.md` deve ser triado depois com o prompt [[PROMPT_TRIAR_INBOX]].

## Regras

- Anote rápido, sem se preocupar com formatação perfeita.
- Quando triar, mover/transformar em uma nota estruturada e **apagar** o arquivo da inbox.
- Itens muito antigos (>30 dias) sem triagem: apagar ou converter em [[OPEN_LOOP]].
